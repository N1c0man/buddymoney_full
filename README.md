# BuddyMoney — Ready-to-Run

## Quick start
1. Install Node LTS from https://nodejs.org (if not already).
2. Open a terminal in this folder and run:
   ```bash
   npm install
   npm start
   ```
3. Open http://localhost:3000

## What’s inside
- React (Create React App)
- Tailwind CSS (preconfigured)
- Framer Motion (animations)
- React Router (multi-page feel)
- Tools:
  - Budget Tracker
  - Savings Goal
  - Debt Payoff
  - Bill Splitter
  - Emergency Fund
  - Net Worth Calculator
